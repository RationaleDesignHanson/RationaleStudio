üéØ Migration Script Features

The automated migration script (scripts/migrate-terminal-gold.js) handles:

  • Tailwind Classes Converts text-[#FFD700], bg-[#FFD700], border-[#FFD700] to token classes
  • Inline Styles Migrates color/backgroundColor/borderColor properties to className utilities
  • SVG Attributes Converts fill="#FFD700" and stroke="#FFD700" to RGB format
  • Box Shadows Converts glow effects to rgba format with appropriate opacity
  • Dry Run Mode Preview all changes before applying with --dry-run flag
  • Statistics Reports files processed, replacements made, and manual review needed
