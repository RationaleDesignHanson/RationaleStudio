## Prompt Optimization Tips:

**Key Style Elements to Maintain:**
- `--style raw` for more realistic, less AI-stylized results
- `--v 6` for latest Midjourney version
- Aspect ratios optimized for different use cases (hero shots 16:9, social media 4:5, detail shots 3:4)
- Consistent color palette: sage green, charcoal gray, cream, mint
- Premium aesthetic references: Apple, Kinfolk, scandinavian minimalism
- Lighting: soft natural light, golden hour, or professional studio lighting

**To Adjust for Different Needs:**
- Add `--chaos 20` for more variation options
- Add `--stylize 250` for more artistic interpretation (or lower number for more literal)
- Replace `--ar` values based on platform (Instagram: 4:5, hero banner: 21:9, etc.)
- Add specific camera references for more photographic control (e.g., "shot on Hasselblad X2D")
